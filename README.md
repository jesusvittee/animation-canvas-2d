## 📌 Descripción del proyecto
Este proyecto consiste en una aplicación web que implementa animaciones 2D utilizando el elemento `<canvas>` de HTML5 y programación orientada a objetos en JavaScript.

Se desarrolló un sistema de burbujas dinámicas que:
- Aparecen desde diferentes direcciones
- Simulan gravedad (caída hacia abajo)
- Rebotan con pérdida de energía
- Desaparecen progresivamente
- Se generan de forma aleatoria

---

## ⚙️ Tecnologías utilizadas
- HTML5
- CSS3 (Glassmorphism + Bootstrap)
- JavaScript (Canvas API)
- Git y GitHub

---

## 🚀 Funcionalidades principales
- Control del número de objetos animados
- Ajuste del tamaño del canvas
- Selección de dirección de entrada
- Animación fluida con `requestAnimationFrame`
- Generación dinámica (spawner aleatorio)
